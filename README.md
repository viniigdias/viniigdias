### 📂 Projetos em Destaque e Estudos de Caso

*   📈 **Simulador Orçamentário:** Sistema completo de gestão financeira com dashboard DRE interativo. Desenvolvido com React, TypeScript e gerenciamento de estado via Zustand.
*   ⚙️ **Aura Manager CRM:** Plataforma de CRM customizada, projetada desde a concepção até a operação para atender demandas complexas de produtores de eventos e agências de viagens.
*   ⚡ **Flashpost (Cache Paradigm):** Implementação e estudo focado no paradigma de cache chave-valor utilizando Redis para otimização extrema de performance.
*   🔒 **Benchmarking de Algoritmos (Criptografia):** Projeto acadêmico de análise de performance (tempo de execução) testando o hash de senhas entre BCrypt e Argon2id.
*   🧪 **Testes de Carga e Monitoramento:** Execução de scripts de estresse contra APIs (Node.js e C#) utilizando k6, com métricas expostas no Prometheus e visualizadas no Grafana.
*   📝 **Quantizap:** Desenvolvimento de aplicação acompanhada de relatório técnico-científico nos padrões metodológicos da UNIFOR.
